# [Bit a Bit]

Bit a Bit é um jogo de trilha com batalhas por turno que representa, de forma lúdica, a jornada acadêmica dentro do curso de Ciência da Computação. A proposta é transformar os desafios da graduação em momentos de estratégia e combate.

Este projeto está sendo desenvolvido como parte do Trabalho Interdisciplinar 4 (TI4), onde devemos aplicar conceitos de grafos e inteligência artificial em um jogo criado com a engine Unity. Nosso objetivo é construir uma experiência interativa que una os conhecimentos adquiridos em diversas disciplinas, utilizando grafos para estruturar os caminhos e mapas do jogo, e IA para criar comportamentos desafiadores e dinâmicos nos inimigos.
