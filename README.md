## 📜 Sobre o Projeto

**Bit a Bit** é um jogo de trilha com batalhas por turno que transforma os desafios da graduação em momentos de estratégia e combate. O jogador explora um mapa progressivo e avança por batalhas de diferentes dificuldades, representando a jornada do estudante.

Este projeto está sendo desenvolvido como parte do **Trabalho Interdisciplinar 4 (TI4)**. Nosso objetivo é construir uma experiência interativa que una os conhecimentos adquiridos em diversas disciplinas, aplicando conceitos de **grafos** para estruturar os caminhos e mapas do jogo, e **Inteligência Artificial (IA)** para criar comportamentos desafiadores e dinâmicos nos inimigos, tudo isso construído na engine Unity.

## ✨ Principais Funcionalidades

- 🗺️ **Mapeamento em Nós (Grafos):** Progressão num mapa estilo árvore (`Branching Paths`). O jogador navega avaliando as opções de avanço por lutas "Fáceis", "Médias" ou o temido "Chefão".
- ⚔️ **Sistema de Turnos e IA:** Uma mecânica consolidada de seleção aliado-inimigo, com inteligência artificial aplicada para ditar as ações e o nível de desafio dos adversários.
- 📐 **Spawns Dinâmicos e Tilemaps:** Posicionamento de personagens regido por Tilemaps Isométricos, possibilitando gerações dinâmicas pelo cenário no começo do combate.
- 💻 **Código Base Limpo e Desencapsulado:** Estrutura modular fazendo uso de interfaces, padronização orientada a objetos (OOP) e Scripting em C#.

## 💻 Como Executar Localmente

- **Pré-requisitos**

    É necessário ter instalado o [Unity Hub](https://unity3d.com/get-unity/download) e uma versão recente da Unity instalada.

- **Rodando o jogo**

    Realize o clone deste repositório para sua máquina local usando:

    ```bash
    git clone [https://github.com/SEU-USUARIO/Bit-by-Bit.git](https://github.com/SEU-USUARIO/Bit-by-Bit.git)
    ```

    1. Abra o Unity Hub, vá até a aba "Projects", e clique em Open.
    2. Navegue até a pasta `Codigo/BitaBit` onde clonou o projeto e a selecione.
    3. Após o Unity carregar as dependências e abrir a interface (pode levar alguns minutos na primeira vez), navegue até a janela Project.
    4. Em `Assets/Cenas`, abra a cena de Mapa (ou Inicializadora) e pressione o botão Play (▶) acima da Game View para iniciar o fluxo.
